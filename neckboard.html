Neckboard

An educational website for guitar and bass players.

Designed and coded by a musician for musicians.
Easily visualize scales and chords across the fretboard of your instrument.
Change the tuning, change the string count to fit your unique needs.

With the ability to visualize 2/3/4 notes per string and the CAGED system patterns.

Technologies used:

  • JavaScript
  • HTML
  • CSS
  • Web Audio API
  • VueJS
